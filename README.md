# Spring Boot practice

A simple Spring Boot application using HTMX+[PicoCSS](https://picocss.com/docs/containers.html) and j2html for the frontend.

## Build

`mvn install`

## Run Tests

`mvn test`

## Run

After build, run:

```bash
java -jar ./target/logit-0.0.1-SNAPSHOT.jar
```
